# Match Platform — Платформа текстовых трансляций 

## О проекте 
Match Platform — это **микросервисная платформа** для текстовых онлайн-трансляций спортивных матчей с обновлением счёта в реальном времени и системой уведомлений. 

---

## Возможности 
- Управление матчами и командами 
- Автоматическое обновление счёта при голах 
- Таймлайн событий + комментарии комментатора 
- Комментарии болельщиков 
- Реакции на события 
- Уведомления подписчикам 
- JWT-аутентификация 

---

## Архитектура  
### Микросервисы

- **Gateway Service** — маршрутизация и JWT
- **User Service** — пользователи и авторизация
- **Match Service** — матчи, команды, игроки, счёт
- **Event Service** — события матча и таймлайн
- **Notification Service** — уведомления
